# Health-Data-Analysis
Analysing Health Data with help of data sets. process is to collet different data sets from real world which consists of patients data such as BMI, cholestrol, glucose levels, blood pressure, diabetes, age, gender, heart rate etc.. starts with collection of data and next data cleaning, next step is to uploaded raw data in python which consists of libraries like Pandas and Numpys then executing it, output is displayed of visualizations like barcharts, graphs, etc.. this is done by Matploitlib/Seaborn.
